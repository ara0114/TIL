# psql
- PostgreSQL 관리 툴
### 전체 DB의 Instance 목록
- \list 또는 \l
### Schema 목록
- \dn
### 접속한 DB의 Table 목록
- \dt
### System table 목록
- \dS
### 테이블의 Column 목록
- \d+ [테이블명]
### 접속한 DB의 Index 목록
- \di
### 접속한 DB의 Function 목록
- \df
### 등록된 사용자 권한 정보 목록
- \du 또는 \dg
### 접속한 DB의 View 목록
- \dv
### 접속한 DB의 Sequence 목록
- \ds
